Аналіз впливу файлу .gitignore
Під час виконання роботи було встановлено, що файл .gitignore фільтрує службові файли Visual Studio.
* Локально: папки bin та obj містять тимчасові файли та результат збірки (.exe), що займають багато місця.
* Віддалено: ці папки відсутні, що робить репозиторій чистим та економить місце на сервері
